# 🌐 Mini Portfolio – Dorline Adelaar

## 📌 Présentation du projet

Ce projet est un mini portfolio web réalisé dans le cadre de mon apprentissage du développement web.

L'objectif est de créer une page simple permettant de présenter mon profil, tout en mettant en pratique les bases du HTML et du CSS.

Ce projet fait partie de ma progression dans le parcours Responsive Web Design de FreeCodeCamp.

---

## 🎯 Objectifs du projet

- Apprendre à structurer une page web avec HTML
- Utiliser CSS pour créer un design simple et lisible
- Comprendre l'organisation d'un projet web
- Adapter l'affichage pour les petits écrans (responsive design)
- Publier et gérer un projet avec Git et GitHub

---

## 🛠️ Technologies utilisées

- HTML5  
- CSS3  
- Visual Studio Code  
- Git  
- GitHub  

---

## 📂 Structure du projet

mini-portfolio/

index.html → structure principale du site  

style.css → styles et mise en page  

images/ → ressources visuelles (logo, images)

README.md → documentation du projet

---

## ✨ Fonctionnalités

- Présentation personnelle
- Carte de profil avec image
- Mise en page simple et claire
- Site adaptable aux petits écrans
- Organisation des fichiers d’un projet web

---

## 📱 Responsive Design

Le site utilise une media query CSS afin d'améliorer l'affichage sur les écrans plus petits :

- réduction de la taille des images  
- adaptation de la largeur des éléments  
- meilleure lisibilité sur mobile  

---

## 🚀 Voir le projet

Pour voir le projet :

1. Télécharger ou cloner le repository
2. Ouvrir le fichier index.html dans un navigateur
